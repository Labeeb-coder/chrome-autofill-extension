# Job Form Autofill Chrome Extension

This Chrome Extension helps users automatically fill common job application fields with one click.

## Features
- Autofills:
  - Full Name
  - Email Address
  - Phone Number
  - Skills / Experience
- Simple popup with Autofill button
- Built using Manifest V3

## Tech Stack
- JavaScript (ES6)
- Chrome Extension APIs
- DOM Manipulation

## Setup Instructions
1. Download or unzip this project.
2. Open Chrome and go to chrome://extensions/
3. Enable Developer Mode.
4. Click Load Unpacked.
5. Select this project folder.

## How to Use
1. Open any job application form.
2. Click the extension icon.
3. Click Autofill Form button.
